# IMOS Frontend

IMOS is the React + Vite + TypeScript frontend for the public product site, launch guidance, and documentation surface.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Clerk

## Local development

```bash
npm install
npm run dev
```

## Branding

The current site is branded as `IMOS` and uses an orange + deep-black visual system across the landing page, docs shell, and product surfaces.
